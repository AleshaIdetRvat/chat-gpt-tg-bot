# Chat GPT telegram bot

## Add .env with tokens for root directory

```bash
OPEN_AI_TOKEN=***********
TELEGRAM_TOKEN=***********

# Optional. In this case, access be only for specified tg-user
TELEGRAM_ID=*********
```

## Install npm packages

```
npm install
```

or if you use yarn

```
yarn
```

## Start bot

```
npm run start
```

or if you use yarn

```
yarn start
```
